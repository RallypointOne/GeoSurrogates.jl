## v0.1.0

Initial release of GeoSurrogates.jl.

### Features
- Classical surrogate models: LinReg, IDW, RBF, TPS, RasterWrap, CategoricalRasterWrap, GeomWrap
- Neural network surrogates: ImplicitTerrain (cascaded SIREN), WindSurrogate (vector fields), CatSIREN (categorical classification)
- AdditiveModel for boosting-style composite surrogates
- Seamless Rasters.jl integration with CRS/extent preservation
- Normalization utilities for preparing data
- Quarto documentation with versioned deployment
